Das Beta-Programm ermöglicht es, neue Funktionen und Features bereits vor dem offiziellen Release zu nutzen und zu testen.

## Zugang

Die Beta-Version ist unter [beta.univelop.de](https://beta.univelop.de) erreichbar. Sie arbeitet auf der **gleichen Datenbank** wie die produktive Version unter app.univelop.de — es entsteht kein Mehraufwand oder Datenverlust durch die Nutzung.

Für die Nutzung auf mobilen Geräten (iOS/Android) muss eine explizite Registrierung für das Beta-Programm erfolgen. Kontaktiere uns dafür unter support@univelop.de.

Admins können in den Einstellungen der Benutzerverwaltung das Beta-Programm für alle Mitglieder des Arbeitsbereichs aktivieren.

## Fehler melden

Falls in der Beta-Version ein Fehler auftritt, kann dieser über den Button **Problem melden** im App-Banner, über das Drei-Punkte-Menü oder per E-Mail an support@univelop.de gemeldet werden.

Folgende Angaben helfen bei der schnellen Bearbeitung:

- **Arbeitsbereich:** z. B. Müller GmbH
- **URL:** z. B. `https://app.univelop.de/#/ws/...`
- **Mitarbeiter und Rolle**
- **Gerät:** Browser / iOS / Android
- **Fehlerbeschreibung inkl. Screenshots**

{: .hint }
Bei Performance-Problemen kann der **Debug-Modus** (über das Drei-Punkte-Menü aktivierbar) nützliche Daten zur Fehlerursache liefern.

## Hinweise

- Es kann vorkommen, dass innerhalb eines Releasezyklus zusätzliche Anpassungen oder kleine Features zum Testen in der Beta freigegeben werden
- Für umfassendes Testen empfehlen wir, Mitarbeitende aus verschiedenen Rollen und Einsatzgebieten einzubinden
- Neue Funktionen sollten nicht direkt in bestehende Prozesse integriert werden, da Nutzer der produktiven Version diese Features noch nicht nutzen können
