Unter **Design** wird das visuelle Erscheinungsbild des Arbeitsbereichs angepasst.

## Bilder

1. **Banner** — das Bild, das über den Kacheln auf dem Homescreen angezeigt wird
2. **Firmenlogo** — wird oben links im Homescreen über dem Banner dargestellt
3. **Hintergrund Anmeldefenster** — das Hintergrundbild der Anmeldeseite im Portal-Modus

## Farbschema

1. **Primärfarbe** — die Hauptfarbe des Arbeitsbereichs, die für Buttons, Links und hervorgehobene Elemente verwendet wird
2. **Sekundärfarbe** — ergänzende Farbe für Akzente
3. **Kachelfarben** — bis zu sechs weitere Farben, die für die Einfärbung einzelner Kacheln genutzt werden können

Die Farben können über eine Farbskala oder durch direkte Eingabe eines HEX-Werts festgelegt werden.
