Univelop bietet Tastenkürzel für häufig verwendete Aktionen auf Desktop-Geräten. Die Übersicht der verfügbaren Kürzel kann in den Arbeitsbereich-Einstellungen unter **Tastenkürzel** eingesehen werden.

## Verfügbare Tastenkürzel

| Tastenkürzel | Aktion |
|---|---|
| `Strg + B` | Zurück navigieren |
| `Strg + H` | Zum Homescreen navigieren |
| `Strg + F` / `Cmd + F` | Suche öffnen |
| `Strg + D` | Duplizieren |
| `Strg + Entf` / `Strg + Backspace` | Löschen |
| `Strg + <` | Arbeitsbereich wechseln |
| `Strg + Shift + ?` | Debug-Modus umschalten |
| `Escape` | Dialog oder Panel schließen |
| Tooltip-Taste | Tooltips anzeigen |
