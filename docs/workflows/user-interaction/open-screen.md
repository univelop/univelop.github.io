Mit dem Schritt _Öffne Seite_ wird auf dem Gerät des Benutzers eine bestimmte Seite geöffnet. Je nach Einstellung kann eine Listenansicht, ein bestehender Datensatz, ein neuer Datensatz, eine Seite oder der Homescreen angezeigt werden.

## Einstellungen

1. **Aktion** — Die Art der Navigation:
   - **Zu Liste navigieren** — Öffnet eine Listenansicht.
   - **Zu Datensatz navigieren** — Öffnet einen bestehenden Datensatz.
   - **Neuen Datensatz erstellen** — Erstellt und öffnet einen neuen Datensatz.
   - **Zu Seite navigieren** — Öffnet eine bestimmte Seite.
   - **Zum Homescreen navigieren** — Kehrt zum Homescreen zurück.
   - **Seite schließen** — Schließt die aktuelle Seite.
2. **Filter und Sortierung** — _Je nach Aktion._ Filter für die anzuzeigende Liste.
3. **Datensatz-ID** — _Je nach Aktion._ Die ID des zu öffnenden Datensatzes.

## Hinweise

- Nur in **Client-Automatisierungen** verfügbar.
- Dieser Schritt verbraucht keine [Credits](/docs/credits).
