Mit dem Schritt _Dupliziere einen Eintrag_ wird eine Kopie eines bestehenden Datensatzes erstellt. Alle Bausteinwerte werden übernommen. Der duplizierte Datensatz ist in folgenden Schritten über den technischen Namen zugreifbar.

## Einstellungen

1. **Verknüpfung mit** — Die Liste, in der der Datensatz liegt.
2. **Datensatz-ID** — Die ID des zu duplizierenden Datensatzes. Kann als Formel angegeben werden.
3. **Parameter** — _Optional._ Variablen, die im duplizierten Datensatz abweichende Werte setzen.

## Hinweise

- Verfügbar in: Client-Automatisierung, Server-Automatisierung, Geschäftsprozess.

## Verwandte Schritte

- [Erstelle Eintrag](/docs/workflows/record-loading/create-record) — Zum Erstellen eines leeren Datensatzes mit definierten Werten
