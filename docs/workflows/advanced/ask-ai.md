Mit dem Baustein _Frag KI_ kann ein Prompt an eine KI gesendet werden. Der Baustein kann nur in Arbeitsbereichen benutzt werden in denen die KI-Features aktiviert sind (Melde dich an [support@univelop.de](mailto:support@univelop.de) um diese aktivieren zu lassen.).

### <span style="color:#0b5394">**Einstellungen**</span>

1. <span style="color:#0b5394">**Integration**</span>  
   Wähle hier eine **OpenAI Integration** aus, welche du in den Arbeitsbereich Einstellungen erstellen und konfigurieren kannst.
1. <span style="color:#0b5394">**Prompt**</span>  
   Gebe hier deinen Prompt an die KI an.
1. <span style="color:#0b5394">**Antwortformat**</span>  
   Wähle hier das Antwortformat aus, welches von der KI zurückgegeben werden soll. Möglich sind Freitext und JSON.
1. <span style="color:#0b5394">**Antwort-Schema**</span>  
   Wenn du JSON als Antwortformat ausgewählt hast, kannst du hier ein [JSON-Schema](https://json-schema.org/overview/what-is-jsonschema) angeben, welches die Antwortstruktur definiert.
