## <span style="color:#0b5394">**Bausteine**</span>

### <span style="color:#0b5394">**Workflow-Buttons in Listenansicht**</span>

Workflow-Buttons lassen sich in der Listenansicht platzieren, um häufig ausgeführte Workflows besser zugänglich zu machen.

## <span style="color:#0b5394">**Workflows**</span>

### <span style="color:#0b5394">**Erfasse Benutzereingabe**</span>

Ermöglicht das Abfragen eines Werts des aktuell angemeldeten Benutzers über Client-Workflows.

### <span style="color:#0b5394">**Frage ChatGPT**</span>

PDFs können an die KI übergeben werden, damit diese analysiert werden und Fragen dazu gestellt werden können.

## <span style="color:#0b5394">**Weitere Änderungen**</span>

-   Verbesserter Import von Datensätzen: Der „Standardstatus" wird nun bei Importen berücksichtigt.
-   Fehler bei der Darstellung des Datensatz-Bausteins behoben.
-   Verbesserte Referenzprüfung beim Löschen von Bausteinen.
