## <span style="color:#0b5394">**Workflows**</span>

### <span style="color:#0b5394">**Der AI Agent ist jetzt im Workflow nutzbar**</span>

Ab sofort kann unser AI-Agent direkt in Workflows eingesetzt werden, ganz ohne manuelle Eingabe.
Damit lassen sich wiederkehrende Aufgaben oder Entscheidungen vollautomatisiert durch den Agenten ausführen. So sparst du Zeit und machst deine Abläufe noch effizienter.

![ask-ai-step](\assets\changelog\1-42\ask-ai-step.png 'ask-ai-step')

## <span style="color:#0b5394">**Verschiedenes**</span>

### <span style="color:#0b5394">**API Zugriffe - Neue „Route" für API-Zugriffe**</span>

Unsere API wurde erweitert: Über den neuen Endpunkt kannst du nun Daten nicht nur bearbeiten, sondern auch neue Datensätze über Sekundärschlüssel anlegen.
Das ermöglicht mehr Flexibilität und neue Integrationsmöglichkeiten für deine individuellen Anwendungsfälle.
