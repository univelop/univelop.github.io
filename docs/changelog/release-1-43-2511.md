## <span style="color:#0b5394">**Kacheln**</span>

### <span style="color:#0b5394">**Vorbelegung bei Seite und Formular**</span>

Mit der neuen Version erhalten die Formular- und Seiten-Kacheln eine neue Einstellung namens „Vorbelegung". Über diese Einstellung kann festgelegt werden, welche Elemente automatisch angezeigt oder ausgefüllt werden, sobald die Seite geöffnet wird. Die Vorbelegung dient somit dazu, die Eingabe zu beschleunigen, indem Formulare und Seiten bereits vorausgefüllt werden.

## <span style="color:#0b5394">**Verschiedenes**</span>

### <span style="color:#0b5394">**Lizenzen - Durchsetzung der Lizenzen**</span>

Mit dem neuen Release werden die gebuchten Lizenzen nun konsequent durchgesetzt. Wenn nicht genügend Lizenzen für alle Nutzer gebucht wurden, werden die verfügbaren Lizenzen strikt angewendet: Nur die Nutzer, für die eine Lizenz gebucht wurde, können arbeiten.

-   Kachel-Import verbessert
-   Terminplaner für den Eingeschränkten Modus verbessert
-   Workflow-Schritt "Öffne Seite" verbessert
