## <span style="color:#0b5394">**Kacheln**</span>

### <span style="color:#0b5394">**Einstellungs Kachel**</span>

Eine neue Kachelart "Einstellungen", die dazu dient häufig verwendete Daten, wie beispielsweise E-Mail Signaturen und E-Mail Adressen, zu verwalten und einfach wiederzuverwenden.

![settings-tile](\assets\changelog\1-40\settings-tile.png 'settings-tile')

## <span style="color:#0b5394">**Workflows**</span>

Bei Univelop wurden bisher Geschäftsprozesse und Automatisierungen in Workflows vermischt. Ab jetzt gibt es eine klare Trennung zwischen beiden. Das sorgt für ein besseres Designerlebnis und bildet die Basis für zukünftige Weiterentwicklungen.

![flow-categories](\assets\changelog\1-40\flow-categories.png 'flow-categories')

### <span style="color:#0b5394">**Geräteseitige Automatisierung**</span>

Wird ohne Wartezeit lokal auf dem Gerät ausgeführt und kann interaktive Schritte, wie das Zeigen von Nachrichten, beinhalten. Das Gerät muss aktiv sein.

### <span style="color:#0b5394">**Cloud-Automatisierung**</span>

Wird in der Cloud ausgeführt. Ideal für systemseitige Aufgaben wie das Senden von E-Mails, die Anbindung externer Systeme oder zeitgesteuerte Aufgaben.

### <span style="color:#0b5394">**Geschäftsprozess**</span>

Ermöglicht die Abbildung längerer Abläufe. Unterstützt Wartezustände auf Entscheidungen, wie das Warten auf eine Genehmigung.

## <span style="color:#0b5394">**AI Agent**</span>

### <span style="color:#0b5394">**AI Agent läuft auf Chat GPT5**</span>

Der AI Agent wurde verbessert! Mit der Weiterentwicklung von Chat GPT durch OpenAI bieten sich neue Möglichkeiten. Auch unser AI Agent läuft ab sofort auf GPT5.
