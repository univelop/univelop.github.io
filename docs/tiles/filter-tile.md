Die _Gefilterte Liste_ zeigt eine vorgefilterte Ansicht einer bestehenden [Einfachen Liste](/docs/tiles/basic-tile). Sie eignet sich ideal, um Teilmengen von Daten gezielt darzustellen — z. B. alle offenen Aufgaben, die eigenen Arbeitszeiten oder ungeprüfte Anträge.

## Erstellen

Eine gefilterte Kachel kann auf zwei Wegen erstellt werden:

1. **Aus der Listenansicht** — In einer geöffneten Liste einen Filter setzen und über das Speichern-Symbol als gefilterte Kachel sichern.
2. **Im Designmodus** — Im [Designmodus des Arbeitsbereichs](/docs/designmode/workspace) eine gefilterte Kachel per Drag & Drop auf den Homescreen ziehen und die Basisliste sowie die gewünschten Filter konfigurieren.

## Einstellungen

Zusätzlich zu den [allgemeinen Kacheleinstellungen](/docs/tiles/general-settings):

1. **Verknüpft mit** — Die Basisliste, deren Daten gefiltert angezeigt werden.
2. **Filter und Sortierung** — Die Filterkriterien, die festlegen, welche Datensätze in der Kachel erscheinen. Die Sortierung bestimmt die Reihenfolge.
3. **Kachel-Info** — Wie bei der Einfachen Liste: Anzahl oder Summe, beeinflusst durch die gesetzten Filter.
4. **Bei einzelnem Datensatz direkt zum Datensatz springen** — Überspringt die Listenansicht bei nur einem Treffer.

## Vorausfüllung bei neuen Einträgen

Wird über eine gefilterte Kachel ein neuer Datensatz erstellt, werden die Filterwerte automatisch in den neuen Eintrag übernommen. Das spart Eingabezeit und reduziert Fehler.

## Verwandte Kacheln

- [Einfache Liste](/docs/tiles/basic-tile) — Die Basiskachel, auf der gefilterte Listen aufbauen
